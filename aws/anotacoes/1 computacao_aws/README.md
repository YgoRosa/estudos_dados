# AWS — Fundamentos de Computação em Nuvem (Teoria)

Este diretório reúne minhas **anotações teóricas em AWS**,
estudando os conceitos desde Cloud Computing básico até Containers.

O foco neste momento é **entender arquitetura, serviços e quando usar cada abordagem**.
A parte prática será adicionada posteriormente na pasta `praticas`.

## Conteúdos estudados
- Cloud Computing e seus benefícios
- Infraestrutura global da AWS (Regions, AZs, Edge Locations)
- Modelos de serviço (IaaS, PaaS, SaaS)
- Provisionamento de recursos
- EC2 e seus componentes
- Elastic Load Balancing
- Auto Scaling
- Serviços de mensageria
- Computação serverless
- Containers na AWS
