Cloud = usar a infraestrutura de outra empresa pela internet.
Não compra servidor, só usa e paga conforme o consumo.

Benefícios principais:
- Elasticidade: aumenta/diminui recursos quando precisa.
- Escalabilidade: aguenta mais usuários fácil.
- Alta disponibilidade: menos chance de ficar fora do ar.
- Custo sob demanda: paga só o que usa.
- Segurança gerenciada: AWS cuida da base, você cuida do seu app.
